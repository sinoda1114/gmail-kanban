# 画面設計書

## 3.1 画面一覧

| 画面         | 内容              |
| ---------- | --------------- |
| ログイン画面     | Googleログイン      |
| 案件カンバン画面   | メイン画面           |
| 案件登録画面     | Gmail本文・URL貼り付け |
| AI整理結果確認画面 | 抽出内容の確認         |
| 案件詳細画面     | 基本情報・面談準備・メモ    |
| 面談準備タブ     | AI面談対策          |
| 面談メモタブ     | 面談中・面談後メモ       |
| 要対応一覧      | リマインド対象         |
| 設定画面       | Googleカレンダー連携など |
| 課金画面       | Stripeプラン管理     |

```text
┌────────────────────────────────────────────┐

```
